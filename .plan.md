## Design Tokens

### Color

| Token | Value | Role |
|-------|-------|------|
| `--bg` | `#0A0A0F` | Page background |
| `--surface` | `#111118` | Card background |
| `--surface-raised` | `#17171F` | Drawer / panel background |
| `--border` | `#2A2A38` | Subtle borders |
| `--text-primary` | `#E8E6F0` | Main text |
| `--text-secondary` | `#8A87A0` | Labels, metadata |
| `--accent` | `#7B68EE` | Medium slate-violet — reveal accent |
| `--accent-glow` | `rgba(123, 104, 238, 0.15)` | Glow behind accent elements |
| `--highlight` | `rgba(123, 104, 238, 0.25)` | Inline text highlight on reveal |
| `--grid` | `rgba(255,255,255,0.03)` | Background grid lines |

### Typography

| Role | Typeface | Weight | Notes |
|------|----------|--------|-------|
| Display | `DM Serif Display` | 400 | Scenario text, model name |
| Body | `Inter` | 300–400 | Definitions, drawer content |
| Label | `JetBrains Mono` | 400 | Type tags, timestamps, metadata |

### Spacing Scale

Follow Tailwind default; key values:
- Card padding: `p-8` (32px)
- Section gap: `gap-6` (24px)
- Drawer section gap: `gap-4` (16px)

### Motion

| Motion type | Duration | Easing |
|-------------|----------|--------|
| Reveal fade | 400ms | `ease-out` |
| Highlight sweep | 600ms | `ease-in-out` |
| Drawer open | 300ms | `spring(stiffness: 300, damping: 30)` |
| Particle ambient | continuous | — |

## Component Specs

### `ScenarioCard`

- Full-width card, max-width `680px`, centered
- Scenario text in `DM Serif Display`, ~22px, generous line height (1.65)
- Scenario type label in `JetBrains Mono`, top-left, secondary color
- State: `idle | revealing | revealed`
- On reveal: key phrases in the scenario text get a highlight sweep (background color transition)

### `RevealButton`

- Centered below scenario text
- Label: "Reveal the model →" (idle) → changes to "Revealed" (done)
- Subtle border, accent color on hover
- Triggers reveal sequence step-by-step with a short delay between steps

### `ModelPanel`

- Appears below RevealButton after step 1 of reveal
- Model name: large, `DM Serif Display`, accent color
- Short definition: `Inter`, secondary color, smaller
- Animated fade-up entry

### `DetailDrawer`

- Collapsible panel below ModelPanel
- Trigger: "Explore further ↓"
- Four sections, each independently expandable:
  - What it is
  - Why it matters
  - Other examples (bulleted)
  - Common misuse
- Smooth height animation using Framer Motion `AnimatePresence`

### `ParticleBackground`

- 40–60 small particles (`2–4px` diameter)
- Subtle drift motion, very slow (not distracting)
- React to cursor proximity — gentle repulsion within `80px` radius
- Color: `--accent` at `0.3` opacity
- Performance: use `requestAnimationFrame` on a `<canvas>` element
- Disable on `prefers-reduced-motion`

### `GridBackground`

- Fixed position, full viewport
- CSS `background-image: linear-gradient(...)` approach for performance
- Dot grid, spacing `~40px`, dot size `1px`
- Color: `--grid`

## Key Decisions Log

| Date | Decision | Rationale |
|------|----------|-----------|
| — | No login/auth | Reduces friction massively; localStorage sufficient for MVP |
| — | Reveal is progressive (3 steps) | Mimics discovery; more engaging than instant reveal |
| — | Static JSON content | Faster to ship; Anthropic API integration deferred to v2 |
| — | Serif display font for scenarios | Signals "reading / contemplation" vs typical app UI |
| — | No filtering/search in MVP | Simplicity; enough to validate with 25 scenarios |
